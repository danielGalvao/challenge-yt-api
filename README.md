# Fictícia Vídeos

## Pré requisitos para teste

- nodejs versões entre 5.0 até 6.8
- npm

## Build do projeto

1. na pasta raiz
2. rode o comando: npm install

## Inicializando projeto

1. rode o comando npm run dev

## Projeto

- rodando em http://localhost:8080
