# Fictícia Vídeos

## Pré requisitos para teste

- nodejs versões entre 5.0 até 6.8
- npm

## Build do projeto

1. Na pasta raiz
2. Rode o comando:
```
$ npm install
```

## Inicializando projeto

1. Rode o comando
```
$ npm run dev
```

## Projeto

- Rodando em http://localhost:8080
